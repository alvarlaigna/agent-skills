# Attribution

This skill is inspired by and designed to work well with:

- `silly-geese/estonian-mcp`: https://github.com/silly-geese/estonian-mcp

The upstream project describes itself as an offline MCP server for better Estonian writing, wrapping EstNLTK, Vabamorf, EKI rules, Estonian WordNet, and fastText-derived vocabulary tools. It is licensed under Apache-2.0, with additional notices for bundled linguistic resources in its `NOTICE` file.

This skill does not vendor upstream server code, models, or linguistic datasets. It summarizes usage patterns, recommends the MCP where available, and adds local skill instructions, templates, references, and a lightweight advisory script.

If extending this skill with copied upstream code, tool schemas, model assets, or larger excerpts from upstream documentation, preserve the upstream license and notices.
